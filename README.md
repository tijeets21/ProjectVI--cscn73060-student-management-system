# Student Management System – CSCN73060

## Overview
This project is a RESTful Student Management Web Server developed as part of CSCN73060 – Performance Engineering.
The system supports CRUD operations using HTTP methods and is designed for performance testing using Apache JMeter.

## Technologies
- C++ with Crow Framework
- SQLite Database
- Docker (Ubuntu)
- RESTful API with JSON
- Apache JMeter

## Features
- Full REST support (GET, POST, PUT, PATCH, DELETE, OPTIONS)
- Dockerized deployment
- Performance testing and optimization
- Simple and accessible web UI

## How to Run
1. Build Docker image
2. Run container
3. Access API endpoints via browser or Postman

## Performance Testing
Apache JMeter is used to simulate concurrent users and analyze response time, throughput, and error rates.
